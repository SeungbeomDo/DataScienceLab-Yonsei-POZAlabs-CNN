## DataScienceLab-Yonsei-POZAlabs-CNN

### 1. 프로젝트 개요
Commu data를 활용하여 코드프로그레션 간 유사도 측정 알고리즘 개발 <br>
<데이터출처: POZAlabs gitub>
https://github.com/POZAlabs 

### 2. Ideation

CNN 알고리즘 적용해 이미지 유사도 측정
